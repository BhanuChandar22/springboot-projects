## Project Overview 
The **Employee Management Application** is a web-based project built using Spring Boot that performs CRUD operations for managing employee data. It supports features like soft deletion, editing employee details, and displaying employees with pagination.
## Features
- Create new employees.  
- Edit existing employee details.  
- Soft delete functionality to retain deleted employee data.  
- Display employees with:  
  - Pagination for easy navigation.  
  - A complete list view for all records. 

## Technologies Used
- **Backend**: Spring Boot (Web, Data JPA)    
- **Frontend**: HTML, CSS, JavaScript, Bootstrap, JSP, JSTL
- **Database**: Oracle
- **Server**: Apache Tomcat
